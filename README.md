Google Data Analytics Capstone Project analyzing behavioral differences between annual members and casual riders.

## Objective

The goal of this analysis is to identify behavioral differences between casual riders and annual members in order to support strategies that convert casual riders into annual members.

## Tools Used

- Google BigQuery
- Python pandas (exploratory inspection)
- Google Sheets (data visualization)
- Jupyter Notebook

## Data Source

Cyclistic historical trip data provided by Motivate International Inc.

https://divvy-tripdata.s3.amazonaws.com

## Key Findings

- Members generate the majority of rides and demonstrate commuting patterns.
- Casual riders take longer trips and ride more frequently during weekends.
- Casual usage shows stronger seasonal variation.
- Member activity peaks during commuting hours.
